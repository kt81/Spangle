Spangle Media Server (WIP)

------

### Media Streaming

- Ingest
  - RTMP
  - SRT
- Web Origin
  - HLS
  - LL-HLS
  - DASH
  - LL-DASH (CMAF Chunked-Transfer)
- Codecs
  - H.264
  - H.265
  - AAC
  - Opus
  - AV1

### Web Console

- Control multiple servers
- Easy monitoring
- Setting editor

- Transcode

### Others

- Full C# (or .NET family)
- High performance
- Highly customizable (plugin, fork, etc...)
- Fail over with very little gap
- Out-of-box timed metadata support
